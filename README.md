This project is a simple Student Portal developed in PHP that allows users to add, update, and delete student records without using any database. All operations are handled temporarily within a single PHP file, and action messages such as added, updated, or deleted are shown in the URL to indicate successful operations. The project uses basic HTML and CSS with a clean color theme and is designed for learning and practice purposes.
